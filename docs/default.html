Remplacez cette page par votre site internet

Vous pouvez envoyer sur le serveur web les fichiers constituant votre site à l'aide d'un programme FTP. Les codes d'accès FTP vous ont été communiqués par email lors de l'ouverture de votre compte d'hébergement.

Lorsque vous vous connectez par FTP sur le serveur, vous vous trouvez directement dans le répertoire web/. C'est dans ce répertoire que vous devez envoyer les fichiers. Le nom de la page d'accueil de votre site doit être index.html (ou index.php). Cette page sera visible sur http://www.mitashi-otha.com.

Gérez votre site et vos comptes emails

Vous disposez d'une console d'administration à l'adresse http://my.oxito.com.

Identifiez-vous grâce au nom d'utilisateur et au mot de passe fournis par Oxito. Depuis la console, vous pouvez entièrement gérer votre hébergement web (comptes emails, utilisateurs FTP, bases de données, statistiques, factures).

Webmail

Vous pouvez consulter vos emails depuis n'importe quel ordinateur connecté à internet grâce au webmail:
http://webmail.mitashi-otha.com.
